                                 nnCron
                                 ~~~~~~

nnCron este un puternic scheduler si reminder, si poate fi folosit ca unealta de automatizari si scripting.

nnCron foloseste un un limbaj propriu de programare (Forth-compatibil) permitand folosirea VBScript/JScript si expresii regulare si ofera extensibilitate prin utilizarea de plug-in'uri.

nnCron intelege formatul tabular 'cron' (Unix) si este administrat prin fisiere 'crontab' usor de editat. Pentru cei care prefera interfetele grafice, nnCorn pune la dispozitie o astfel de interfata care permite adaugarea, modificarea, stergerea si lansarea de taskuri, adaugarea de evenimente cat si configurarea nnCron.

Pe langa functiile traditionale (lansare de aplicatii, documente si mesaje la ore specificate), nnCron poate de asemenea sa:

  - lanseze orice program ca serviciu
  - permite lansarea unui task sub identitatea oricarui utilizator
  - relanseaza taskuri si reminderi sarit
  - opreste si suspendeaza activitatea calculatorului sau trece in "sleep mode" la momentul indicat
  - reactiveaza calculatorul pentru a rula un task
  - afiseaza, ascunde, inchide, termina, minimizeaza, maximizeaza sau ascunde in bara de sistem o fereastra specificata
  - afiseaza mesaje sau le scrie in fisiere jurnal
  - opereaza cu clipboard, fisiere si registrul windows
  - emuleaza activitatea tastaturi si a mouse'ului
  - (de)conecteaza prin dialup
  - foloseste difuzorul sistemului pentru a emite semnale audio de lungimea si frecventa specificata
  - ruleaza fisiere audio
  - sincronizeaza ora sistemului
  - schimba prioritatea unui proces
  - termina orice proces
  - reporneste automat in caz de eroare fatala
  - supravegheaza fisiere, ferestre, procese, activitatea mouse'ului, perioadele de inactivitate, shortcuts, (de)conectari la internet, inserare de dischete in unitatea floppy, prezenta unui host in retea (ping), spatiu disponibil pe disc, etc., etc.
  

nnCron este un produs robust si usor de utilizat. Ruleaza extrem de rapid si ofera o multitudine de facilitati dezvoltate in colaborare cu alti utilizatori nnCron.

Cerinte de sistem, instalare/dezinstalare nnCron
---------------------------------------
  - IBM PC sau compatibl
  - Procesor Intel Pentium sau o varianta superioara
  - Windows 95/98/ME/NT/2000/XP

Pentru a instala nnCron, lansati fisierul de distribute, destinatia instalarii facandu'se in timpul instalarii.
Pentru dezinstalarea nnCron, folositi 'Add or remove programs' din 'Control Panel'.
Setarile nnCron se afla in fisierul 'nncron.ini'.

---------------------------------------
Lansarea si oprirea nnCron
---------------------------------------
Cel mai usor de lansat manual serviciul nnCron este prin rularea 'startnncron.bat' din directorul de baza al nnCron. Oprirea nnCron se face prin rularea 'stopnncron.bat' sau click pe 'Exit' din meniul nnCron (disponibil prin dreapta-click pe icoana nnCron din bara sistem).

Shortcuts catre fisierele mentionate se gasesc in grupul de programe nnCron, din meniul de start.
